# LiveLikeView
仿钉钉直播点赞动画
[参考文章](https://www.jianshu.com/p/e6034c2bc1ac)
### 添加使用依赖
```
allprojects {
     repositories {
           ...
          maven { url 'https://jitpack.io' }
     }
}
```
### 引入项目
```
dependencies {
	implementation 'com.github.wangjianxiandev:LiveLikeView:v1.0.0'
}
```
#### 注意：该版本的点赞图标出现位置有些不正确，后续版本将会更正，也期待大家提出建议
